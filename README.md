# Personal Finance Ledger

A command-line interface (CLI) application for managing personal finances using dictionaries for storage and strict type hinting.

## Features

- Add income and expense transactions
- View transaction history with filtering options
- Generate financial summaries
- Calculate current balance
- Export/import data to/from JSON
- Colorful console output using Rich

## Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd personal-finance-ledger

2. Install dependencies:
pip install -r requirements.txt
```

## Starting Up the Ledger:
```
python ledger.py
```

## Available Commands:
    add: Add a new transaction
    view: View transactions
    summary: Show financial summary
    balance: Show current balance
    export: Export data to JSON
    import: Import data from JSON
    delete: Delete a transaction
